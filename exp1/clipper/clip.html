    Instructions

  1. Observe the circuit diagram of the Clipper.
  2. Click on the Power Button .
  3. Select the Amplitide(A) of the input sine wave signal(Vi).
  4. Select the frequency of the signal(f) for the input signal(Vi).
  5. Select the battery voltage(Vr)
  6. Observe the input signal(Vi) and clipped output signal(Vo) in the graph.
  7. Change the values of A, f and Vr to observe the variation in the output signal.
